# TableFlow Control Web

Standalone platform control web for TableFlow super admins.

## Runs On

- `http://localhost:3001`

## Capabilities

- Sign in as super admin
- Manage tenant customers
- Onboard new restaurants
- Review all platform users
- Enable or disable system users

## Setup

```bash
cp .env.example .env.local
npm install
npm run dev
```
